.. _glossary:

========
Glossary
========

There are some terminology which is used through thte documentation,
and it is strictly related to astronomical image processing techniques,
and implementation details.


.. glossary::

    source
        A light source, often used to name stars and galaxies in images.

    covariance matrix
        The matrix :math:`C` formed such as  :math:`C_{ij} = <o_i, o_j>`,
        where :math:`<,>` is the inner product operator,
        and each :math:`o_i` is an observation.
